CD             Displays the name of or changes the current directory.
HELP           Provides Help information for Datagen commands.
CREDITS        Shows the people who made Datagen.
QUIT           Quits the program.
SOURCE         Downloads the latest source code from the Github repository.

RELOADLANG     Reloads the language files.
